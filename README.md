# Notação pré-fixa com ANTLR4
Converte uma expressão matemática simples no formato infixa para o formato pré-fixa, utilizando uma gramática do ANTLR4.<br>
Exemplo:<br>
<pre>
1+1 => +11
(1+2)*3  => *+123
3*(1+2)  => *3+12
</pre>
